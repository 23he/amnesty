Using the Amnesty menu

The Amnesty menu, which appears as a butterfly icon in the top-right area of your system's menu bar, is the main user interface for Amnesty Widget Browser.

When launched, Amnesty loads widgets from the system's widget libraries and builds a list of the widgets it finds. If no widgets are found, then the list will consist of a single disabled menu item reading No widgets found.

The Amnesty menu also contains items that you'd find in a standard application's menu bar, such as Help, Preferences, Quit and other application functionality.


Installing a widget for use in Amnesty

For Panther users, the Amnesty widget library is the main widget library and the location where widgets must be installed.

For Tiger users, the Amnesty widget library is an optional location where "Amnesty only" widgets can be installed.

In the Mac OS X Finder, open the Amnesty widget library (~/Library/Application Support/Amnesty/Widgets) or press the Show In Finder button in the Preferences window to open this location automatically.

Using the Finder, you can install .wdgt files (or remove them) from this location.

Special Notes for Panther users:

Some widgets are written specifically to use technologies available only in Tiger. These widgets, when loaded in Amnesty under Panther, may not run at all. Mesa Dynamics maintains a web page of Known Issues, including a list of widgets known not to run in Panther. This page is accessible via the Known Issues menu item under the Help submenu. If you are unsure whether a widget is supposed to run in Panther or not, you can check this page to make sure.

Apple's default Dashboard widgets (which are included with Tiger) are unsupported (and inaccessible) in Amnesty running in Panther.


Refreshing the Widget List

If you install (or remove) widgets while Amnesty is running, you can force the widget list to be updated at any time.

Choose Refresh Widget List from the Amnesty menu.
