Trucs et astuces avancés

Utilisation d'Amnesty avec un sous-ensemble des widgets Dashboard (Tiger seulement)

Si vous utilisez seulement Amnesty avec un groupe choisi de widgets, vous pouvez vouloir installer des copies de ces widgets dans la bibliothèque de widgets Amnesty. Ensuite, désactiver le chargement des bibliothèques de widget Dashboard Utilisateur et Dashboard par défaut dans les Préférences. Ceci maintiendra votre menu Amnesty beaucoup plus court, et libérera ces widgets que vous ne voulez pas utiliser (ou qui ne fonctionnent pas) dans Amnesty.

Groupement de widgets dans des sous-menus

Si vous avez une grande bibliothèque de widgets et que votre menu Amnesty devient trop long, vous pouvez vouloir grouper vos widgets dans la bibliothèque de widgets d'Amnesty dans des sous-dossiers. Chaque sous-dossier dans cet emplacement apparaîtra comme un sous-menu sous le menu d'Amnesty (dans Tiger, voici comment le sous-menu Widgets échantillon est implémenté).

Auto-installation de widgets (Panther seulement)

Pour installer automatiquement des widgets dans la bibliothèque d'Amnesty, glissez-déposez des fichiers .wdgt sur l'icône de l'application Amnesty (ou double-cliquez sur les fichiers .wdgt dans le Finder. Nota : si Amnesty fonctionne déjà, vous devrez également actualiser la liste de widgets après l'installation automatique.

Création de multiples copies d'un widget

Amnesty n'a pas de soutien intégré pour de multiples copies d'un widget, ce qui signifie que vous ne pouvez pas avoir plus d'une copie du widget Horloge mondiale, par exemple. Cependant, vous pouvez faire fi de cette limitation avec un peu de travail :

    1. Faites une copie du fichier .wdgt du widget (pour les utilisateurs de Tiger, il est préférable d'installer cette copie dans la bibliothèque des widgets d'Amnesty pour qu'il n'apparaisse pas dans Dashbord).

    2. Contrôle-cliquez la copie du fichier .wdgt et sélectionner Afficher le contenu du paquet du menu contextuel qui apparaît. Une fenêtre du Finder s'ouvrira affichant le contenu du paquet du widget.

    3. Double-cliquez le fichier Info.plist dans cette fenêtre. Il devrait ouvrir dans Property List Editor d'Apple.

    4. Dans l'éditeur, cliquez sur le triangle en regard du mot Root pour révéler le contenu du fichier.

    5. Modifier la valeur de la clef CFBundleIdentifier pour qu'elle soit unique. Par exemple, modifier "com.apple.widget.worldclock" en "com.apple.widget.worldclock2".

    6. Fermez l'éditeur et lancez Amnesty. Vous devriez avoir maintenant deux widgets distincts.

Le soutien intégré pour de multiples widgets sera ajouté à une version future d'Amnesty.

Utilisation d'Exposé avec Amnesty

Si vous avez mis des widgets sur le niveau de la fenêtre du Bureau, vous pouvez utiliser le mode Bureau d'Exposé (F11) pour les révéler rapidement. De façon similaire, vous pouvez utiliser le mode Toutes les fenêtres d'Exposé (F9) pour faciliter la recherche de ceux-ci dans le niveau standard des fenêtres.

Revenir à la disposition d'origine d'un widget

Si un widget est fermé, vous pouvez revenir à la disposition d'origine (Emplacement et Configuration) en tenant la touche Option enfoncée en sélectionnant le widget à partir du menu Amnesty. À l'ouverture, les réglages de la disposition reviendront aux réglages d'origine. Ceci fonctionne aussi en sélectionnant une disposition à partir du sous-menu Disposition (tous les widgets fermés dans cette disposition auront leurs réglages remis à zéro.)

Ouverture d'Amnesty en mode sans échec (Safe Mode)

Par défaut, Amnesty se rappelle la dernière disposition ouverte et l'ouvre la prochaine fois qu'elle est lancée. Si pour quelque raison, un widget dans cette disposition fait planter ou geler Amnesty, vous pouvez forcer Amnesty à se lancer avec tous les widgets fermés en tenant la touche Majus enfoncée au démarrage.

Remettre les préférences internes d'un widget manuellement

La plupart des widgets ont leurs propres préférences internes "arrière", ceux-ci sont employés pour sauvegarder et restaurer certains réglages. Dans certains cas, vous pouvez vouloir enlever ces préférences pour "remettre à zéro" un widget qui ne fonctionne pas correctement :

    1. Dans Amnesty, contrôle-cliquez un widget et choisissez Lire les Infos à partir du menu local qui apparaît.

    2. Prenez note du champ Identifiant.

    3. Fermer widget.

    4. Dans le Finder, ouvrez le dossier de préférence des widgets d'Amnesty (~/Bibliothèque/Application Support/Amnesty/Preferences).

    5. Déplacez le fichier dont le nom correspond à celui de l'Identifiant du widget à la corbeille.

    6. Dans Amnesty, ouvrez le widget. Celui-ci devrait être remis à zéro.

Noter que ceci diffère de remettre à zéro les réglages de la disposition à zéro.
