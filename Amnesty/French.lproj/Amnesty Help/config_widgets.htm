Configuration d'un Widget

Contrôle-cliquer un widget et choisir Configurer depuis le menu local qui apparaît.

Fenêtre

    Opacité. L'opacité d'un widget : de presque invisible à complètement opaque.

    Niveau. Le niveau dans lequel le widget est placé : Flottant (au-dessus de toutes les autres fenêtres), Standard (au même niveau que toutes les autres fenêtres) and Bureau (sous toutes les autres fenêtres, imbriqué dans le bureau).

    Verrouiller en place. Locks the widget into its current location in the layout. Verrouille le widget à son emplacement actuel dans la disposition.

    Ignorer les clics. Place le widget dans un mode "clic au travers" où les clics de souris passent au travers. Choisir Ramener la disposition à l'avant dans le sous-menu Disposition pour ignorer ce réglage.

Avancé

    Mise à jour automatique. Si le widget est conçu pour se mettre à jour lorsqu'affiché, ce réglage force le widget à se mettre à jour à l'intervalle spécifié.

    Taille. La taille du widget : de 50 % à 150 %, en pas de 25 %.

    Rotation. La rotation du widget : de 0 à 360 degrés du centre du widget.

Nota : les widgets qui sont redimensionnés ou tournés peuvent ne pas répondre à la souris uniformément une fois cliqués, particulièrement lorsque déplacés. Selon vos réglages, vous pouvez devoir découvrir un "point d'attache" que vous pouvez employer pour déplacer un widget avec votre rotation et/ou taille particulière.
