Utilisation du menu Amnesty

Le menu Amnesty, qui apparaît comme une icône de papillon dans la zone de droite de la barre des menus de votre système, est l'interface utilisateur principale d'Amnesty Widget Browser.

Une fois lancé, Amnesty charge les widgets à partir des bibliothèques du système et construit du widget une liste des widgets qu'il trouve. Si aucun widget n'est trouvé, alors la liste ne se composera d'un seul d'élément de menu intitulé Aucun widget trouvé.

Le menu Amnesty contient également les éléments que vous trouveriez dans la barre des menus d'une application standard, tels que Aide, Préférences, Quitter et autres fonctionnalités de l'application.


Installation d'un widget pour l'utiliser dans Amnesty

Pour les utilisateurs de Panther, la bibliothèque de widgets d'Amnesty est la bibliothèque principale des widgets et l'emplacement où les widgets doivent être installés.

Pour les utilisateurs de Tiger, la bibliothèque des widgets d'Amnesty est un emplacement facultatif où des widgets "Amnesty seulement" peuvent être installés.

Dans le Finder de Mac OS X, ouvrez la bibliothèque de widgets d'Amnesty (~/Bibliothèque/Application Support/Amnesty/Widgets) ou appuyez sur le bouton Afficher dans le Finder dans la fenêtre de Préférences pour ouvrir cet emplacement automatiquement.

En utilisant le Finder, vous pouvez installer des fichiers .wdgt (ou les supprimer) de cet emplacement.

Notes spéciales pour des utilisateurs de Panther :

Ces widgets, une fois chargés dans Amnesty sous Panther, peuvent ne pas fonctionner du tout. Mesa Dynamic maintien à jour une page Web des problèmes connus, y compris une liste de widgets connus qui ne fonctionnent pas sous Panther. Cette page Web est accessible par l'intermédiaire de l'élément de menu Problèmes connus sous le sous-menu Aide. Si vous êtes incertain qu'un widget est censé fonctionner ou non sous Panther, vous pouvez regarder sur cette page pour vous en assurer.

Les widgets par défaut de Dashboard d'Apple (qui sont inclus avec Tiger) ne sont pas supportés, ni accessibles dans Amnesty fonctionnant sous Panther.


Actualiser la liste des Widgets

Si vous installez (ou retirez) des widgets tandis que Amnesty fonctionne, vous pouvez forcer la liste des widgets à se mettre à jour à tout moment.

Choisir Actualiser la liste des Widgets du menu Amnesty.
