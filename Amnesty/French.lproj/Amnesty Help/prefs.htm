Préférences

Choisir Préférences depuis le menu Amnesty.

Widgets

Un widget peut se rappeler des réglages suivants dans chaque disposition où il est présent :

    Visibilité. Si le widget est visible ou non.

    Emplacement. Où le widget réside sur l'écran lorsqu'il est visible.

    Configuration. Les différents réglages disponibles dans la fenêtre de configuration du widget (opacité, niveau, etc.).

Une fois lancée, Amnesty charge des widgets à partir d'un maximum de trois bibliothèques, dans l'ordre suivant (si le même widget est trouvé dans plus d'un emplacement, le premier widget trouvé est utilisé et les autres sont ignorés) :

    Amnesty. C'est l'endroit standard pour la bibliothèque de widgets d'Amnesty. Sous Tiger, les widgets installés dans cet emplacement ne sont pas chargés par Dashboard.

    Dashboard Utilisateur. C'est l'endroit où Dashboard installe des widgets tiers. Cette bibliothèque n'est pas accessible sous Panther.

    Dashboard par défaut. C'est l'endroit où Dashboard installe les widgets par défaut écrits par Apple. Cette bibliothèque n'est pas accessible sous Panther.

Général

    Exécuter en mode de consommation réduite. Quelques widgets "très actifs" (e.g. ceux avec une animation constante) ont été assumés par leurs auteurs pour fonctionner seulement quand Dashboard était en activité et ainsi n'ont pas été conçu nécessairement pour réduire au minimum leur utilisation des ressources du système. Ce mode aide à garder ces widgets sous contrôle, ce qui peut être utile s'ils sont toujours en marche et visibles.

    Fermer les widgets automatiquement lorsque masqué. Afin de réduire à un minimum l'utilisation de la mémoire, vous pouvez préférer fermer automatiquement tout widget qui n'est pas visible.

    Lancer Amnesty avec tous les widgets fermés. Cette option (dans laquelle Amnesty se lance avec la disposition masquée active) est utile si vous avez défini Amnesty pour qu'il se lance pendant la procédure de connexion et vous souhaitez maintenir les widgets masqués jusqu'à ce que vous les appeliez explicitement (e.g. utilisant une touche raccourci ou le menu Amnesty).

Raccourcis

Les raccourcis sont des combinaisons de touches clavier qui lancent la fonctionnalité dans Amnesty à tout moment, n'importe l'application qui fonctionne.

    Masquer/afficher la disposition. C'est équivalent à choisir Masquer ou Afficher à partir du sous-menu Disposition et basculera la visibilité de la disposition actuelle.

    Ouvrir le menu Amnesty (Tiger seulement). C'est équivalent de cliquer sur l'icône du papillon de menu d'Amnesty : le menu s'ouvre et les éléments sont sélectionnables par l'intermédiaire de la souris, des touches de déplacement du curseur ou en utilisant "entrée prédictive" (e.g. en tapant "cal" pour choisir le widget de calendrier et puis appuyer sur return pour l'ouvrir).

Nota : Les modifications faites aux raccourcis entrent en vigueur la prochaine fois que Amnesty est lancée.
