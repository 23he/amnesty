Travailler avec des Widgets

Widgets sont des mini applications qui sont conçues pour fonctionner dans l'environnement Dashboard de Mac OS X Tiger. En utilisant Amnesty Widget Browser, ces mêmes widgets peuvent fonctionner sur le bureau, avec des capacités augmentées, complètement en dehors de l'environnement Dashboard (en fait, Dashboard n'est pas nécessaire pour utiliser ou même être installer pour qu'Amnesty fonctionne).

Cette "indépendance" de Dashboard permet aussi aux utilisateurs de Mac OS X Panther (10.3.9) d'exécuter et d'apprécier plusieurs widgets tiers dans Amnesty avant de mettre à jour pour Tiger.

Les widgets dans Amnesty sont accessible via le menu Amnesty. La façon dont un widget est listé dans le menu dépend de son état :

    Désactivé. Les widgets qui ne sont pas actuellement accessibles sont affichés en gris. Un widget dans cet état ne peut pas être choisi. Si un widget manque une information importante pour pouvoir l'exécuter sans risque dans Amnesty, ou s'il est officiellement non supporté, alors il apparaîtra toujours dans cet état.

    Fonctionnant et visible. Les widgets qui fonctionnent actuellement et sont également visibles sont affichés avec l'icône d'une coche en regard de leur nom. Un widget dans cet état utilise du temps de traitement (processeur) et de la mémoire.

    Fonctionnant et masqué. Les widgets qui fonctionnent actuellement et sont masqués (invisible à l'écran) sont affichés avec l'icône d'un tiret en regard de leur nom. Un widget dans cet état utilise de la mémoire (même si cette mémoire est typiquement "paginée" et donc pas un fardeau sur le système).

    Fermé. Widgets qui sont fermés (i.e. ni fonctionnant, ni visibles) sont affichés sans aucune icône. Un widget dans cet état n'utilise aucune ressource du système.

Quand un widget fonctionne et est visible vous pouvez agir sur celui-ci en cliquant sur tous les contrôles à l'intérieur de son cadre. Ou, si vous cliquez sur une partie statique du widget, vous pouvez glisser le widget dans l'écran et le placer où vous désirez.

Conseil : Si pour quelque raison vous ne pouvez pas utiliser avec les commandes du widget, cliquez le widget une fois et alors appuyez et tenez enfoncé la touche Commande tout en cliquant le widget. De même, si vous ne pouvez pas trouver un endroit "pour saisir" le widget pour le déplacer, cliquez sur le widget une fois et alors appuyez et tenez la touche option enfoncée tout en cliquant le widget.


Ouverture d'un widget

Choisir le nom d'un widget fermé depuis le menu Amnesty.

L'icône de papillon Amnesty indiquera que le widget se charge. Quand le widget est prêt, il apparaîtra sur votre écran (centré sur l'écran principal si vous n'avez pas ouvert ce widget précédemment).


Masquage d'un widget

Choisir le nom d'un widget fonctionnant et visible depuis le menu Amnesty; ou Contrôle-cliquez un widget et choisissez Masquer du menu contextuel qui apparaît.

Le widget disparaîtra de l'écran.


Afficher un widget masqué

Choisir le nom d'un widget fonctionnant et masqué depuis le menu Amnesty.

Le widget apparaîtra à l'écran.


Fermeture d'un widget

Contrôle-cliquez un widget et choisissez Fermer le widget du menu contextuel qui apparaît.

Le widget disparaîtra de l'écran et se fermera.


Mise à jour d'un widget

Dans Dashboard, beaucoup de widgets qui exigent accès à Internet ou affichent un certain type d'état du système mettront à jour leur information chaque fois que Dashboard est affiché (par opposition à la mise à jour à un intervalle fixe).

Avec Amnesty, vous pouvez décider de maintenir un de ces widgets visible à tout moment, et en conséquence, le widget se mettra à jour seulement la première fois qu'est affiché. Dans ces cas, vous pouvez forcer le widget à mettre à jour son affichage à tout moment.

Contrôle-cliquez un widget et choisissez Mettre à jour du menu contextuel qui apparaît.

Si le widget est conçu pour se mettre à jour lorsqu'il est affiché, il le fera.
